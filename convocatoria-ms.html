CICLO ESPECIALIZADO DE INNOVACCION VIRTUAL

Ago 01, 2021


Gracias al plan Innovar por México, Microsoft trae el programa Innovacción Virtual que te ofrece obtener la certificación internacional AZ-900 de Microsoft Azure Fundamentals de forma totalmente gratuita. Misma certificación que te da mejores opciones de empleabilidad.

El ciclo consta de 4 semanas iniciando el 13 de agosto de 2021 con 3 ejes temáticos:

  • Ciencias de datos
  • Inteligencia Artificial
  • Arquitectura de soluciones
Beneficios
  • Certificado internacional Microsoft
  • 100 dólares en créditos de Azure
  • Github student developer pack(Canva pro, dominios, etc)
Requisitos
  • Ser estudiante de educación superior
  • Pertenecer a una institución publica de mexico 
  • Disponer de dos horas diarias para estudiar y certificarte

El ciclo especializado es totalmente gratuito al igual que las herramientas que aquí se ofrecen.
